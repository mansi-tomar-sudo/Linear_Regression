# Medical Cost Prediction using Linear Regression

## Project Overview
This project predicts individual medical insurance costs using the Linear Regression machine learning algorithm. It analyzes factors such as age, BMI, number of children, smoking status, sex, and region to estimate medical insurance charges.

## Dataset
- Source: Kaggle
- Number of rows:1338
- Number of columns:7

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
- Data Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Handling Missing Values (if any)
- Categorical Data Encoding
- Feature Selection
- Train-Test Split
- Linear Regression Model Training
- Model Evaluation
- Medical Cost Prediction

## Results
The Linear Regression model predicts medical insurance costs based on demographic and health-related features. The model's performance is evaluated using metrics such as R² Score, MAE, MSE, and RMSE to measure prediction accuracy.
