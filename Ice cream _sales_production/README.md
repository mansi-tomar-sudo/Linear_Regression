# Ice Cream Sales Prediction using Linear Regression

## Project Overview
This project predicts ice cream sales using the Linear Regression machine learning algorithm. It analyzes the relationship between temperature and ice cream sales to estimate future sales based on weather conditions.

## Dataset
- Source: Kaggle
- Number of rows:577
- Number of columns:2

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
- Data Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Data Visualization
- Feature Selection
- Train-Test Split
- Linear Regression Model Training
- Model Evaluation
- Sales Prediction

## Model Evaluation
- R² Score:0.059018751022322014
- Mean Absolute Error (MAE):23.44830724666072
- Mean Squared Error (MSE):796.7886489260718
- Root Mean Squared Error (RMSE):28.22744495922491

## Results
The Linear Regression model predicts ice cream sales based on temperature. The model's performance is evaluated using metrics such as R² Score, MAE, MSE, and RMSE to measure prediction accuracy.
